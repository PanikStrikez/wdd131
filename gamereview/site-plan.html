Game Review Site Plan

Ethan Dalebout

WDD 131-A1

Overview

Purpose

My site helps gamers discover, review, and track video games across platforms. It provides personalized suggestions, trending titles, and community ratings.

Audience

The primary audience includes PC and console gamers aged 16 35 who want an interactive hub for game recommendations, quick reviews, and wish lists.

Dynamic elements

- Random Game of the Day - ES module (randomGame.js) exports a function to pick a random game object from a JSON array. - Listens for a “Surprise Me” button click, then injects the games image and info into the DOM. - Genre & Platform Filters - Uses array methods (filter, map) to narrow games by genre or platform. - Dropdown change events trigger re-rendering of the game grid. - Favorites List - Each game card has an “Add to Favorites” button. Clicking pushes the game object into a favorites array. - Saves and retrieves this array from localStorage. - Toggles button text and style based on whether the game is already favorited (conditional branching). - Game Detail View - On clicking “View Details,” updates the URL hash and dynamically imports a detail module (detailView.js). - Module fetches the appropriate game data and populates a main section with images, description, ratings, and user comments. - User Ratings & Comments - Star-rating widget that updates a game object s ratings array. Uses reduce to calculate average score. - Comment form listens for submit events, validates input, then appends new comment to the DOM and to the games comments array.
